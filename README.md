# PyTorch Learning Journey

This repository documents my progress in learning PyTorch and deep learning through hands-on exercises and small experiments.

## Learning Goals

- Understand tensors, shapes, data types, and devices
- Build neural networks with `torch.nn`
- Write training and evaluation loops
- Work with `Dataset` and `DataLoader`
- Evaluate models with appropriate metrics
- Save, load, and run trained models
- Apply transfer learning to a practical problem
